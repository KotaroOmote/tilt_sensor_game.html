傾きでボールをゴールへ！

使用したセンサーと目的

このWebページでは、スマートフォンの 加速度・角速度センサー を利用するために DeviceOrientationEvent API を使用しています。
スマホの傾きを検出し、その値（beta: 前後の傾き、gamma: 左右の傾き） に応じて青いボールを移動させます。
プレイヤーはスマホを傾けながらボールを操作し、障害物を避けて赤いゴールに入れることで ポイントを獲得します。
これにより、人間の動作（デバイスの傾き）をリアルタイムにセンシングし、 インタラクティブな体験を実現しています。
API 詳細: MDN Web Docs
